# Amato Music

## 概要
9日間のハッカソン型サマーインターンで作成して優勝した、SpotifyAPIを用いて音楽を再生しながらチャットができるWebサービス

## 使用技術
フロントエンド: React, Redux  
バックエンド: Node.js  
データベース: MongoDB

## 開発人数
フロントエンド: 3人  
バックエンド: 1人
